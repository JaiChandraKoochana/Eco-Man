# Eco-Man
Eco-Man: Clean the Planet is a Pac-Man clone built with Python/Pygame, reskinned around an environmental theme. You play "Eco-Man" (a recycler) navigating a classic Pac-Man-style maze, collecting pollutants instead of dots, while avoiding "ghosts" reskinned as pollution sprites (Blinky/Pinky/Inky/Clyde reused as-is).
